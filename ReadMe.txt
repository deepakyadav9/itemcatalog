This application is all about adding restaurants and their menu items.
Authentication and Authorization is being done, protocol used is OAuth2.0

Required libraries are Python, flask, SQLAlchemy. 
All templates are in templates folder.
static files are in static folder.
google client_id and client_secret is in client_secrets.json file.
facebook app_id and app_secret is in fb_client_secrets.json file.

setup database by using command python database_setup.py
then populate it by using command python lotsofmenus.py
run project.py from current directory by using command python project.py.
